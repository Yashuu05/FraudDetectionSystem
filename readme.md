### About

- This is a Fraud detection system which uses machine learning to predict if the transaction is fraud or not.
- The system consist of 3 layers: 
    1. Z-score
    2. Unsupervised layer
    3. Supervised layer

---

### Dataset : 
source : https://www.kaggle.com/datasets/sriharshaeedala/financial-fraud-detection-dataset

---

### Algorithms used: 

1. Z-score
2. Isolation Forest (anaomaly detection)
3. XGB classifier
